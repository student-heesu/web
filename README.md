# 🎵 감정 음악 추천 웹사이트

기분에 따라 감정을 선택하면 배경 색깔이 달라지고 음악을 추천해주는 인터랙티브 웹사이트입니다.

## ✨ 주요 기능

### 1. 히어로 섹션
- 50개의 이모지가 화면에서 자유롭게 떠다니는 애니메이션
- 6가지 다양한 애니메이션 패턴
- 매력적인 그라데이션 배경

### 2. 감정 선택
- 7가지 감정 옵션:
  - 😊 행복 (Happy)
  - 😢 슬픔 (Sad)
  - 😠 화남 (Angry)
  - 😌 평온 (Calm)
  - 🤩 신남 (Excited)
  - 😰 불안 (Anxious)
  - 🥰 사랑 (Love)


### 3. 동적 배경 변경
- 각 감정에 맞는 색상 그라데이션으로 부드럽게 전환
- 감정별 테마 색상 적용

### 4. 음악 추천
- 선택한 감정에 어울리는 여러곡의 음악 추천
- 곡명, 장르 정보 제공
- 카드 형식으로 시각적 표현
