# 🎵 감정 음악 추천 웹사이트

기분에 따라 감정을 선택하면 배경 색깔이 달라지고 음악을 추천해주는 인터랙티브 웹사이트입니다.

## ✨ 주요 기능

### 1. 히어로 섹션
- 50개의 이모지가 화면에서 자유롭게 떠다니는 애니메이션
- 6가지 다양한 애니메이션 패턴
- 매력적인 그라데이션 배경

### 2. 감정 선택
- 8가지 감정 옵션:
  - 😊 행복 (Happy)
  - 😢 슬픔 (Sad)
  - 😠 화남 (Angry)
  - 😌 평온 (Calm)
  - 🤩 신남 (Excited)
  - 😰 불안 (Anxious)
  - 🥰 사랑 (Love)


### 3. 동적 배경 변경
- 각 감정에 맞는 색상 그라데이션으로 부드럽게 전환
- 감정별 테마 색상 적용

### 4. 음악 추천
- 선택한 감정에 어울리는 6곡의 음악 추천
- 곡명, 아티스트, 장르 정보 제공
- 카드 형식으로 시각적 표현

## 🚀 사용 방법

### 1. 파일 열기
- `index.html` 파일을 웹 브라우저에서 엽니다.
- 또는 Live Server를 사용하여 로컬 서버로 실행합니다.

### 2. 웹사이트 사용
1. 히어로 섹션에서 떠다니는 이모지를 감상합니다.
2. 아래로 스크롤하거나 화살표를 클릭합니다.
3. 현재 기분과 어울리는 감정 버튼을 클릭합니다.
4. 자동으로 스크롤되며 배경색이 변경됩니다.
5. 추천된 음악 리스트를 확인합니다.

## 🔧 커스터마이징

### 감정 추가하기
`scripts/emotionData.js` 파일에서 `emotions` 배열에 새로운 감정 객체를 추가하세요.

```javascript
{
    id: 'your-emotion',
    name: '감정이름',
    emoji: '😀',
    description: '설명',
    bgGradient: 'linear-gradient(...)',
    music: [
        {
            title: '곡명',
            artist: '아티스트',
            genre: '장르'
        }
    ]
}
```

### 스타일 변경하기
`styles/main.css` 파일의 CSS 변수를 수정하여 색상, 간격, 폰트 등을 변경할 수 있습니다.

### 이모지 개수 변경하기
`scripts/emojiAnimation.js` 파일의 `initEmojiAnimation` 함수에서 루프 횟수를 변경하세요.
