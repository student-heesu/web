# 감정 음악 추천 웹사이트 TODO

## 📋 프로젝트 개요
기분에 따라 감정을 선택하면 배경 색깔이 달라지고 음악을 추천해주는 웹사이트

## 🎯 핵심 기능
1. **히어로 섹션**: 50개의 이모지가 떠다니는 애니메이션
2. **감정 선택**: 다양한 감정 버튼 제공
3. **동적 배경**: 감정에 따라 배경색 변경
4. **음악 추천**: 선택한 감정에 어울리는 음악 리스트 표시

## 🎨 감정 데이터 예시

### 추천 감정 리스트
1. **행복 (Happy)** 😊
   - 배경: 밝은 노란색 → 오렌지 그라데이션
   - 음악: 경쾌한 팝, 댄스 음악

2. **슬픔 (Sad)** 😢
   - 배경: 회색 → 진한 파란색 그라데이션
   - 음악: 감성적인 발라드, 어쿠스틱

3. **화남 (Angry)** 😠
   - 배경: 어두운 빨간색 → 검은색 그라데이션
   - 음악: 강렬한 록, 힙합

4. **평온 (Calm)** 😌
   - 배경: 연한 파스텔 블루 → 민트 그라데이션
   - 음악: 앰비언트, 클래식, 재즈

5. **신남 (Excited)** 🤩
   - 배경: 비비드한 핑크 → 보라색 그라데이션
   - 음악: EDM, 업비트 팝

6. **불안 (Anxious)** 😰
   - 배경: 어두운 보라색 → 회색 그라데이션
   - 음악: 차분한 피아노, 명상 음악

7. **사랑 (Love)** 🥰
   - 배경: 로즈 핑크 → 레드 그라데이션
   - 음악: R&B, 로맨틱 발라드

8. **에너지 (Energetic)** ⚡
   - 배경: 밝은 그린 → 옐로우 그라데이션
   - 음악: 운동 음악, 파워풀한 곡
